Cobalt Notebook example

This page coordinates multiple independent cobalt note editors. Press Ctrl+Enter inside a note to split it into two notes and move the cursor to the start of the new next note. Press Backspace at the start of a note to join it with the previous note, or Delete at the end to join it with the next note. Arrow keys also move between notes at note boundaries.

Normal Enter still inserts a line break inside the current note. Hold Shift while pressing an arrow key to extend a visual selection across note boundaries.
